ServiceHub Setup Instructions
============================

STEP 1: Install Node.js
-----------------------
1. Go to https://nodejs.org/
2. Download the LTS version (recommended)
3. Run the installer
4. Make sure "Add to PATH" is checked
5. Restart your command prompt after installation

STEP 2: Verify Node.js Installation
-----------------------------------
Open a new command prompt and run:
node --version
npm --version

If these commands work, you're ready to proceed!

STEP 3: Install Dependencies
----------------------------
Run these commands in your project directory:

npm install
cd client
npm install
cd ..

STEP 4: Create Environment File
------------------------------
Create a file named ".env" in the root directory with this content:

MONGODB_URI=mongodb://localhost:27017/servicehub
JWT_SECRET=your-super-secret-jwt-key-change-this-in-production
PORT=5000
NODE_ENV=development

STEP 5: Start the Application
-----------------------------
Run this command:
npm run dev

This will start both the backend (port 5000) and frontend (port 3000)

STEP 6: Access the Application
-----------------------------
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000

STEP 7: Test the Application
---------------------------
1. Open http://localhost:3000 in your browser
2. Register a new account
3. Explore the platform features

TROUBLESHOOTING
===============

If you get "node is not recognized":
- Node.js is not installed or not in PATH
- Restart your command prompt after installing Node.js

If you get "npm is not recognized":
- Node.js installation didn't include npm
- Reinstall Node.js with the official installer

If you get port errors:
- Make sure ports 3000 and 5000 are not in use
- Run: npx kill-port 3000 5000

If you get MongoDB errors:
- The app will work without MongoDB for basic testing
- For full functionality, set up MongoDB Atlas (free)

NEXT STEPS
==========
1. Install Node.js from nodejs.org
2. Run the commands above
3. Open http://localhost:3000
4. Start using ServiceHub!

For MongoDB setup (optional):
- Go to mongodb.com/atlas
- Create a free account
- Create a cluster
- Get your connection string
- Update MONGODB_URI in .env file
